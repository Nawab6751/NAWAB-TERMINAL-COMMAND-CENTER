NAWAB ZADA MASTER COMMAND HUB

Complete Termux & Linux Reference

1. FUNDAMENTAL BASICS
apt update && apt upgrade -y
System ko update karne ke liye.
termux-setup-storage
Internal storage ka access lene ke liye.
ls
List files: Folder ki files dekhne ke liye.
cd [folder_name]
Change Directory: Kisi folder mein jane ke liye.
2. ESSENTIAL TOOLS INSTALL
pkg install python python2 -y
Python scripts chalane ke liye install karein.
pkg install git php curl wget -y
GitHub aur files download karne ke zaroori tools.
3. ADVANCED & PENTESTING
pkg install nmap -y
Nmap: Port scanning aur network scanning ke liye.
pkg install metasploit -y
Metasploit: System penetration aur exploitation ke liye.
git clone https://github.com/sqlmapproject/sqlmap
SQLmap: Database vulnerabilities scan karne ke liye.
4. FILE PERMISSIONS
chmod +x *
Saari files ko "chalne ke qabil" (executable) banane ke liye.

NAWAB ZADA COMMAND HUB

Official Team Reference Guide v3.0

BASIC FUNDAMENTALS
apt update && apt upgrade -y
System aur packages ko update karne ke liye.
Level: Beginner
ls -la
Saari files (chupi hui files samait) ki list detail ke saath dekhna.
Level: Beginner
ADVANCED TOOLS & NETWORKING
pkg install nmap -y
Network Scanner install karne ke liye (Ports check karna).
Level: Advanced
nmap -sV [IP_ADDRESS]
Target IP ke open ports aur un par chalne wali services scan karna.
Level: Advanced
git clone [URL]
GitHub se koi bhi tool ya script download karna.
Level: Intermediate
find / -name "config.php"
Poore system mein specific file dhoondne ke liye.
Level: Professional
